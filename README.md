It is about having a virtual pet. 
Create a Python program that simulates a Tamagotchi-style virtual pet. 
Your program should use a base class with at least two subclasses that inherit from it, and the different pet types should behave differently in some way. 
Beyond that, the design is up to you, choose whatever attributes, methods, and interactions you think make for a fun and well structured program. 
All the comments are included in the code for clarity.
